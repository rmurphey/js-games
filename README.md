# js-games

Prerequisites:

- Node 4.2

Set up the project:

- `npm install`

Start the server:

- `npm start`

## Games

Once the server is running, you can see the following games:

- [number-guessing](http://localhost:8080/number-guessing/)
- [word-guessing](http://localhost:8080/word-guessing)
- [canvas-repl](http://localhost:8080/canvas-repl)
